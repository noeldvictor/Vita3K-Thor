# L3 R3 OSD Toggle - 2026-05-12 12:36:00

## Goal

Add an alternate runtime OSD opener for Windows controller testing: press L3 + R3 together.

## Change

- Added L3 + R3 chord detection to the existing runtime gamepad hotkey path.
- The chord toggles the runtime OSD open/closed only while a title is running.
- The chord is latched until either stick-click button is released, so holding both buttons does not repeatedly toggle the OSD.
- Single L3 and single R3 remain normal Vita controller buttons.

## Notes

This keeps Android Back as the primary Thor OSD opener and keeps Select reserved as the modifier for fast-forward and quickstate hotkeys.
